Техническое Задание: Игра “Сапер” на Pygame с БД рекордов и уровнями сложности

Цель: Разработать игру “Сапер” с графическим интерфейсом на Python с использованием Pygame,
 с реализацией таблицы рекордов в базе данных и тремя уровнями сложности.

Функционал:

Игровое поле
Три уровня сложности:
Легкий: 9x9 клеток
Средний: 16x16 клеток
Сложный: 30x16 клеток
Случайное расположение мин на основе уровня сложности.
Клетки с минами и числами.
Логика:
Открытие клеток, проигрыш при мине.
Автоматическое открытие пустых областей.
Отметка флажками.
Победа при открытии всех не-минных клеток.
Таймер: Подсчет времени игры.
Интерфейс:
Отображение игрового поля, счетчика мин, таймера.
Кнопка “Начать новую игру”.
Меню выбора уровня сложности.
Сообщения о победе/поражении.
Таблица рекордов:
Отображение лучших времен для каждого уровня сложности, полученных из БД.
Возможность просмотра таблицы рекордов.
База Данных:
Хранение имени игрока, уровня сложности, и времени прохождения игры.
Выбор СУБД: SQLite 
Функции для добавления, чтения и сортировки записей рекордов.

Библиотеки:

Pygame: Для графики и управления игрой.
Random: Для генерации случайных чисел.
sqlite3 (или другая библиотека для выбранной СУБД): Для работы с базой данных.
