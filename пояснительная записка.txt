Пояснительная записка: Игра “Сапер” на Pygame с БД рекордов и уровнями сложности

Функционал:

Игровое Поле:

Реализовано игровое поле с тремя уровнями сложности:
“Легкий”: 9x9 клеток. 10 мин на поле
“Средний”: 16x16 клеток. 40 мин на поле
“Сложный”: 30x16 клеток. 99 мин на поле
Минные поля генерируются случайно на основе выбранного уровня сложности.
На игровом поле отображаются клетки с минами и числами, представляющими количество соседних мин.
Также добавлен пользоваетльский режим, в котором пользователь сам устанавливает размеры поле и количество мин на поле.
(высота и ширина от 5 до 40; количество мин на поле от 1 до произведения ширины и высоты - 1)

Логика:

Проигрыш в случае открытия клетки с миной.
Автоматическое открытие соседних пустых областей при открытии пустой клетки.(рекурсивная функция)
Возможность установки флажков для отметки предполагаемых мин.
Победа при открытии всех неминованных клеток.
При поражении демонстрируются неверно отмеченные мины.

Таймер:

Реализован подсчет времени игры от ее начала до завершения (победа или поражение).

Интерфейс:

Графическое отображение игрового поля, счетчика мин и таймера.
Кнопка “Новая игра” для запуска нового и игрового сеанса при выбранной сложности.
Кнопка “Изменить сложность” для возвращения пользователя в главное меню с выбором сложности.
Вывод сообщений о победе или поражении после окончания игры.
При желании пользователя при победе возможность ввести свое имя для таблицы рекордов

Таблица Рекордов:

Обеспечена возможность просмотра таблицы рекордов с сохраненными данными.
Данные отсортированы по уровню сложности и отсортированы по времени(от меньшего прохождения к большему)
База Данных:

Реализовано хранение данных в базе данных SQLite.
Сохраняются имя игрока, уровень сложности, время и дата прохождения игры.
Реализованы функции для добавления, чтения и сортировки записей рекордов.

Библиотеки:

Использована библиотека Pygame для отрисовки графического интерфейса и обработки событий.
Использована библиотека Random для генерации случайных чисел (расположение мин).
Использована библиотека sqlite3 для работы с базой данных SQLite.
Использована библиотека time для измерения времени игры.
Использована библиотека os для работы с путями к файлам.
Использована библиотека datetime для отслеживания даты.
Использована библиотека sys для увеличения глубины рекурсии.
